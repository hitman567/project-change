# Project Change - 60-Day Transformation Tracker

A self-hosted learning tracker for a 60-day MAANG + AI/ML career transformation.

**Live:** [https://hitman567.github.io/project-change/](https://hitman567.github.io/project-change/)

## Features
- Dashboard with day counter, streak, heatmap, progress bars
- Daily 5AM-11AM schedule with Claude Code prompts
- 8-week curriculum: DSA, System Design, AI/ML, Projects
- Task tracker with LeetCode problem logging
- Quiz system (DSA, System Design, AI/ML, Python)
- Morning recap with spaced repetition
- Project roadmap (8 curriculum + custom)
- Curated resources (NeetCode, ByteByteGo, HuggingFace, etc.)
- Journal with history
